# ESP32-C3/S3 プロトタイピング基板 / ESP32-C3/S3 Prototyping board

ESP32-C3やESP32-S3を搭載した、プロトタイピングや一品物の製作に便利な基板です。WROOMモジュールの動作に必要な回路とユニバーサル基板エリアがセットになっているため、マイコンと周辺回路が1枚の基板にスマートに収められます。
